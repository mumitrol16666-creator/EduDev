# Maestro AI Consultant Project Pack

Пример проектного prompt pack для школы Маэстро.

Это не часть универсального ядра. Для подключения укажите:

```bash
AI_CONSULTANT_PROJECT_ID=maestro
AI_CONSULTANT_PROJECT_PROMPT_DIR=backend/examples/ai-consultant/maestro/prompts
AI_CONSULTANT_KNOWLEDGE_DIR=backend/src/modules/ai-consultant/knowledge
AI_CONSULTANT_TRIAL_SLOTS_FILE=backend/src/modules/ai-consultant/trialSlots.json
```

Чтобы сделать проект под другую деятельность, скопируйте эту папку и замените тексты в `prompts/*.md`.
